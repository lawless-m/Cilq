# Extension Icons

Place your extension icons here:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

For now, you can create simple placeholder icons or use any PNG images.

## Quick Icon Creation

You can create simple placeholder icons using any image editor or online tools like:
- https://www.favicon-generator.org/
- https://www.canva.com/

Or use this SVG as a base and convert to PNG:

```svg
<svg width="128" height="128" xmlns="http://www.w3.org/2000/svg">
  <rect width="128" height="128" fill="#4CAF50"/>
  <text x="64" y="80" font-family="Arial" font-size="60" fill="white" text-anchor="middle">CB</text>
</svg>
```

Save as:
- icon16.png (resize to 16x16)
- icon48.png (resize to 48x48)
- icon128.png (keep at 128x128)
